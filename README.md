# VU Kornelija-Dambrauskaite v3.0 – naudojimosi instrukcija

## Diegimas
1. Atsisiųskite `setup.exe` iš GitHub Releases (v3.0).
2. Paleiskite `setup.exe`. Patvirtinkite UAC (Administrator).
3. Programa bus įdiegta į:
   C:\Program Files\VU\Kornelija-Dambrauskaite\
4. Į Start meniu rasite:
   Start -> VU -> Kornelija-Dambrauskaite
5. Ant darbalaukio sukuriama nuoroda.

## Paleidimas
- Spauskite nuorodą „Kornelija-Dambrauskaite“ Start meniu arba darbalaukyje.
- Programos meniu:
  1) Generuoti naują failą – įveskite failo pavadinimą ir kiekį.
  2) Naudoti esamą failą – įveskite esamo .txt kelią.
  3) Įvesti duomenis ranka – suveskite studentų duomenis.
- Pasirinkite:
  - Galutinio balo skaičiavimą: vidurkis / mediana / abu
  - Konteinerį: vector / list
  - Rikiavimą: vardas / pavardė / galutinis
  - Strategiją: A+B / tik vargšiukai / optimizuota
- Rezultatai: kataloge `results\` – `vargsiukai.txt` ir `kietuoliai.txt`.

## Pašalinimas
- Valdymo skydas -> Programs and Features -> „VU Kornelija-Dambrauskaite“ -> Uninstall
- Arba Start -> VU -> „Pašalinti VU Kornelija-Dambrauskaite“.
