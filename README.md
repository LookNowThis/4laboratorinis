# VU Kornelija-Dambrauskaitė v3.0 – naudojimosi instrukcija

## Diegimas
1. Atsisiųskite `setup.exe` iš GitHub Releases (v3.0).
2. Paleiskite `setup.exe` ir patvirtinkite UAC (Administrator).
3. Programa bus įdiegta į:  
   `C:\Program Files\VU\Kornelija-Dambrauskaite\`
4. Start meniu rasite:  
   **Start → VU → Kornelija-Dambrauskaite**
5. Ant darbalaukio bus sukurta nuoroda.

## Paleidimas
Programą galite paleisti:
- iš Start meniu pasirinkę **Kornelija-Dambrauskaite**, arba  
- spustelėję darbalaukio nuorodą.

## Programos meniu
### Galimi veiksmai:
- **Generuoti naują failą** – įveskite failo pavadinimą ir įrašų kiekį.
- **Naudoti esamą failą** – nurodykite esamo `.txt` failo kelią.
- **Įvesti duomenis ranka** – suveskite studentų duomenis.

### Parametrai:
- **Galutinio balo skaičiavimas:** vidurkis / mediana / abu  
- **Konteineris:** `vector` / `list`  
- **Rikiavimas:** vardas / pavardė / galutinis balas  
- **Strategija:** A+B / tik „vargšiukai“ / optimizuota

### Rezultatai
Rezultatai bus išsaugoti kataloge `results\`:
- `vargsiukai.txt`
- `kietuoliai.txt`

## Pašalinimas
Programą galite pašalinti:
- **Control Panel → Programs and Features → „VU Kornelija-Dambrauskaite“ → Uninstall**, arba  
- **Start → VU → „Pašalinti VU Kornelija-Dambrauskaite“**.
